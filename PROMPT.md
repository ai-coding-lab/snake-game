# 要求仕様: スネークゲーム

## 概要

ブラウザで遊べるクラシックなスネークゲームを作成

## 機能要件

### 必須機能

1. ゲーム要素
   - スネーク（頭と体）
   - エサ（ランダム出現）
   - ゲームフィールド

2. 操作
   - 方向転換（←↑→↓キー）

3. ゲームロジック
   - エサを食べるとスネークが伸びる
   - 壁または自分の体に当たるとゲームオーバー
   - エサを食べるとスコア加算

4. 表示
   - スコア表示
   - ハイスコア表示（ローカルストレージ保存）

### あれば嬉しい機能

1. スピード調整（レベルアップで加速）
2. グリッド表示のオン/オフ
3. 壁抜けモード（反対側から出現）

## 非機能要件

- 対応ブラウザ: Chrome最新版
- フレームワーク依存なしのバニラJS
- ホスト側の8888番ポートで接続

## 技術指定

- HTML / CSS / JavaScript（バニラJS）
- 外部ライブラリなし
- Canvas ベースの描画
