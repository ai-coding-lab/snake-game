# Snake Game

ブラウザで遊べるクラシックなスネークゲーム。Canvas + バニラ JavaScript で実装。

## クイックスタート

```bash
docker compose up -d
```

ブラウザで http://localhost:8888 にアクセス。

## 操作方法

- **矢印キー** または **WASD**: 方向転換
- **スペース**: ゲーム開始 / リスタート
- **タップ操作**: モバイル対応ボタン

## 機能

- スコア＆ハイスコア（ローカルストレージ保存）
- レベルアップでスピードアップ
- グリッド表示の切り替え
- 壁抜けモード（Wrap Mode）

## 停止

```bash
docker compose down
```
